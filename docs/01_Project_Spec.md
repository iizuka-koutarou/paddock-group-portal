# PADDOCK GROUP Portal 仕様書

## 1. プロジェクト名

PADDOCK GROUP Portal

## 2. コンセプト

社員が毎朝最初に開くページ。

出社、退勤、お知らせ確認、Academyでの学習を一つの場所で行える社内ポータルシステム。

## 3. Version 1.0 の目的

まずはHTML / CSS / JavaScriptで、社内運用できる画面構成を完成させる。

将来的には Firebase / Next.js へ移行し、ログイン、勤怠、管理機能を本格的にデータベース化する。

## 4. 主な機能

### ログイン

* PADDOCK GROUP Portal として表示
* 既存の背景画像を活用
* 入力欄は大きく、見やすくする
* 初回ログイン時は歓迎画面を表示

### HOME

* 出社ボタン
* 退勤ボタン
* お知らせ
* Academyへの導線
* 前回の続きから学習
* 学習ロードマップ

### 勤怠

* 出社記録
* 退勤記録
* 本人による履歴確認
* 訂正申請
* 管理者による確認
* 管理者による承認 / 却下

### お知らせ

* 全店舗共通のお知らせのみ表示
* 管理画面から追加・編集できるようにする

### Academy

* 共通教育
* 共通スキル
* ブランド教育
* テスト一覧

### 管理画面

* お知らせ管理
* 勤怠確認
* 訂正申請確認
* 教材管理
* 社員管理

## 6. デザイン方針

* 高級感
* 清潔感
* 余白を大切にする
* 毎日使っても疲れない
* 迷わない操作
* ブランドカラーはブランドページのみ使用

## 7. ブランドカラー運用

### 共通画面

白、黒、グレー、シャンパンゴールドを基調にする。

### Kawasaki画面

Kawasaki Greenをアクセントとして使用。

### Ducati画面

Ducati Redをアクセントとして使用。
