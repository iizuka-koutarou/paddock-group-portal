# Sprint 2 - PADDOCK GROUP Portal HOME

## 内容

このSprintでは、Portalとして毎日開くHOME画面の土台を追加します。

## 追加・更新ファイル

- index.html
- dashboard.html
- assets/css/login.css
- assets/css/portal.css
- assets/js/portal.js

## 機能

- ログイン画面の高級感ブラッシュアップ
- Portal HOME
- 出社・退勤ボタン
- 本日の勤務表示
- 訂正申請モーダル
- お知らせ表示
- 次回イベント表示
- Academy入口
- 店舗情報
- 管理画面デモ

## 注意

Version 1.0では localStorage を使ったデモ保存です。
管理者が全員分を確認する本格運用には、Version 2.0でFirebase等のデータベース連携が必要です。
