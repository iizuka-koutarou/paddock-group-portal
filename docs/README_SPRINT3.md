# Sprint 3 更新内容

## 概要
PADDOCK GROUP Portal のログイン画面とHOME画面を、Version 1.0の基準に近づけるために全面ブラッシュアップしました。

- `index.html`
- `dashboard.html`
- `assets/css/login.css`
- `assets/css/portal.css`

## 主な変更

### ログイン画面

- 既存の背景画像は維持
- PADDOCK仮ロゴ表記を大きく配置
- ガラス調ログインカードを高級感のあるデザインへ調整
- 入力欄を大きく、毎日使いやすいサイズへ変更

### Portal HOME

- Academy中心のHOMEからPortal HOMEへ変更
- 出社・退勤・勤務状況を最上部に配置
- 全社共通のお知らせを表示
- 次回イベントを表示

  - Kawasaki Plaza 焼津
  - Ducati 静岡
- Academy入口を配置
- 店舗情報を表示
- 管理画面デモ枠を配置

### 勤怠

- 出社・退勤をブラウザ内に保存
- 勤務時間を自動計算
- 訂正申請モーダルを追加
