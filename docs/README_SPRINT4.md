# Sprint 4 更新内容

## 目的
PADDOCK GROUP Portal の HOME を「毎朝開くポータル」として磨き込みました。

## 主な変更

- HOMEから店舗情報セクションを削除
- HOMEではメーカー系ロゴを使用しない方針に変更
- PADDOCKロゴを共通ロゴとして利用
- 現在時刻を秒までリアルタイム表示
- 日付を表示
- 時間帯によって挨拶を自動変更
  - 朝：おはようございます。
  - 昼：こんにちは。
  - 夕方：お疲れ様です。
  - 深夜：遅くまでお疲れ様です。
- 今日の一言を時間帯・季節を考慮して表示
- 静岡県の週間天気を気象庁データから取得する枠を追加
- 出社・退勤時刻を秒まで記録
- 勤務時間を自動計算
- 訂正申請モーダルを維持
- イベントは合同 / Kawasaki Plaza 焼津 / Ducati 静岡の3枠を維持
- Academyへの導線を整理

## ロゴ運用

- Portal共通画面：PADDOCK GROUP Portalのみ
- Kawasaki / Ducati の正式ロゴは Academy ブランド教育画面で使用予定

## 補足

天気情報は気象庁の公開データを参照します。ブラウザ側で外部データを取得するため、通信環境やブラウザ設定により取得できない場合があります。その場合は気象庁へのリンクを表示します。
