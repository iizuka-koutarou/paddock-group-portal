# 1. Purpose

PADDOCK Portalは社員が毎朝最初に開く業務ポータルである。

社員が一日に必要な情報を一画面で確認できることを目的とする。

Portalは情報を見るだけではなく、

・仕事を始める

・予定を確認する

・会社からのお知らせを確認する

・自分の目標を確認する

・学習へ進む

ための入口となる。
